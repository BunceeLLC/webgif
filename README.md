## Installation
```bash
yarn global add webgif || npm i -g webgif
```

## Usage
```bash
$ webgif https://giphy.com/search/lol -d 60
```
This will navigate to `https://giphy.com/search/lol` and make an animated GIF of duration 60 seconds.
